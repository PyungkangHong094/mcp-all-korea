# mcp-all-korea

한국 특화 MCP 서버 큐레이션 리스트 (awesome-list). 메인 산출물은 `README.md`, 서버별 상세 문서는 `servers/`.

## 하네스: Korea MCP Catalog

**목표:** README의 모든 MCP 서버 링크를 방문해 도구·설치·API 키·설정 예시를 수집한 상세 카탈로그(`servers/*.md`)를 생성·검증·게시한다.

**트리거:** 카탈로그 구축·갱신·재실행·부분 재수집·신규 서버 상세 문서 추가·게시 관련 작업 요청 시 `korea-mcp-catalog` 스킬을 사용하라. 단순 질문(서버 몇 개야? 등)은 직접 응답 가능.
